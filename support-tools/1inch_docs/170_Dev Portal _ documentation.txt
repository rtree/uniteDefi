Web3 RPC API
POST
https://api.1inch.dev/web3/{chainId}/{nodeType}
Perform RPC calls
Performs JSON-RPC calls on the blockchain identified by the given chainId. If nodeType is not provided, a default nodeType will be used.
Network
Ethereum
Parameters
chainId
string
*
(path)
The unique identifier for the blockchain.
nodeType
string
*
(path)
The node type you require for your call. If not provided, 'full' will be used as the default value.

Content type
application/json

const axios = require("axios");

async function httpCall() {
  const url = "https://api.1inch.dev/web3/1/{nodeType}";

  const config = {
    headers: undefined,
    params: {},
    paramsSerializer: {
      indexes: null,
    },
  };
  const body = {
    jsonrpc: "2.0",
    method: "eth_blockNumber",
    id: 1,
  };

  try {
    const response = await axios.post(url, body, config);
    console.log(response.data);
  } catch (error) {
    console.error(error);
  }
}
