Migration from v5.2 to v6.0

Tip: If you are using the fusion-sdk, make sure you are using the latest version: fusion-sdk releases

Following changes must be done to migrate from v5.2 to v6.0.

Parameter change for /quote
Response

Renaming of parameter:
fromToken -> srcToken
toToken -> dstToken
toAmount -> dstAmount

Parameter change for /swap
Added new parameter optional query params:

usePermit2: boolean - use Permit2 for approval

Response:

Renaming of parameter:
fromToken -> srcToken
toToken -> dstToken
toAmount -> dstAmount
